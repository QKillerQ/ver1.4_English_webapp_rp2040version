üìã Setup Instructions

  1. Click "Connect" and select your HIDtaiko device (Chrome/Edge browser required)
  2. Click "Default Values" to load recommended starting values
  3. Adjust sliders to tune sensitivity and timing for your drums
  4. Click "Save To Device" to save settings permanently to device
  5. Use "Load Last Settings" to reload your saved settings anytime

üí° Tip: Sensitivity values are thresholds - lower numbers = higher sensitivity. Start with defaults and adjust gradually.

üìñ Parameter Guide

üéØ Sensitivity (A0-A3): Threshold values for piezo sensors. When sensor voltage exceeds this value, a key press is triggered. Lower values = more sensitive to light taps, higher values = requires harder hits.
‚è±Ô∏è Delay Setting #1 (Same Pad Lockout): Prevents the same pad from triggering multiple times rapidly. Increase if you get unwanted double-hits on the same pad. Usually 1-5ms is sufficient.
‚è±Ô∏è Delay Setting #2 (Surface Pad Lockout): Lockout time for surface pads after any pad is hit. Prevents multiple surface hits but don't set too high or rapid surface hits won't register. Default: 10ms.
‚è±Ô∏è Delay Setting #3 (Rim Pad Lockout): Lockout time for rim pads after any pad is hit. Similar to Delay Setting #2 but specifically for rim pads. Usually slightly higher than surface delay. Default: 30ms.
‚è±Ô∏è Delay Setting #4 (Cross Lockout): Cross-lockout between surface and rim pads. Prevents rim from triggering when surface is hit and vice versa. Critical for preventing false triggers. Default: 32ms.
‚è±Ô∏è Delay Setting #5 (Hold Adjustment): Special timing adjustment for certain rhythm games. Use 0 for most games. Try 20ms if controller works in text editors but has issues in specific rhythm games.
