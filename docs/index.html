説明
  • 感度
    • これからは感度の調節です。これらの値は閾値になっていて、センサーからの電圧が一定を超えた場合にキー入力を行う様になっています。 したがって、値が低いほど感度が低くなります。 多重が起きる箇所は感度を上げましょう。
  • Bdelay
    • これはキーが入力されてから次の入力を受け付けるまでの時間です。この値が大きいほど多重が起きにくくなります。しかし、大きすぎると連打が伸びなくなるので注意してください。 標準:12 単位:ms
  • Cdelay
    • これは先ほどのBdelayの縁だけ版です。説明は同じです。 標準:30 単位:ms
  • Ddelay
    • これは面が入力されてから、縁の入力を制限する時間です。面を叩いたときに縁が多重してしまう場合に値を上げてください。 標準:35 単位:ms
  • Hdelay
    • これは公式アプリや一部のシュミレーターで必要な入力制限です。「メモ帳ではキー入力ができるけど、シュミレーターでは入力ができない」このような場合に値を設定してください。 標準:18 単位:ms
